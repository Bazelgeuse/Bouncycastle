# Bouncycastle
Bouncy Castle 은 암호화에 사용되는 API 모음입니다. Java 및 C # 프로그래밍 언어 모두에 대한 API가 포함되어 있습니다.
Bouncy Castle은 두 명의 동료가 서버 측 Java SE 에서 작업하는 작업을 변경할 때마다 일련의 암호화 라이브러리를 다시 작성해야하는 것에 지쳐 있었기 때문에 시작되었습니다. 
개발자 중 한 명이 자바 ME (당시 J2ME) 개발에서 취미로 활동했으며 설계 고려 사항은 J2ME의 Java VM을 포함하여 라이브러리에 가장 많은 범위의 Java VM을 포함시키는 것이 었습니다. 
이 디자인 고려 사항은 Bouncy Castle에 존재하는 아키텍처로 이어졌습니다.
원래 Java API는 테스트 코드 및 J2ME, JCE / JCA 공급자 및 기본 X.509 인증서 생성에 대한 지원을 포함하여 약 27,000 줄의 코드로 구성되었습니다. 이에 비해 1.53 릴리스는 테스트 코드를 포함하여 390,640 줄의 코드로 구성됩니다. 
PKCS # 10, PKCS # 12, CMS, S / MIME , OpenPGP , DTLS , TLS , OCSP , TSP , CMP , CRMF, DVCS , DANE , EST 와 같은 많은 기능을 갖춘 원본 릴리스와 동일한 기능을 지원합니다.  
C # API는 약 145,000 줄의 코드이며 Java API의 대부분을 지원합니다.
프로젝트 주요 속성
1. 표준 준수 및 적응성에 중점을 둡니다.
2. Bouncy Castle 웹 사이트에 나열된 관련 API에 대한 자료로 상업적 지원을 제공한다.

#JAVA Provider Framework
이 소프트웨어는 주로 자바를위한 프로그래밍 환경으로, 학생들이 복잡한 개발 환경의 기능보다는 프로그램 디자인에 더 집중할 수 있도록 돕습니다. 이 소프트웨어는 강사와 학습자 간의 중개자 역할을합니다. 
이 소프트웨어는이 소프트웨어 내의 다양한 구성 요소와 툴바가있는 사용자 인터페이스를 통해 Java 프로그래밍 언어를 쉽게 실행하고 이해할 수있게 해줍니다. 한편, 우리는 사용자가 자신의 프로그램 요구 사항에 따라 Java 프로그래밍 언어에서 사용되는 다양한 기능의 코드를 
신속하게 생성 할 수있게하여 Java 코드 작성에 익숙해 지도록 돕고 자합니다.
JAVA 프로그래밍 언어를 선택하는 주된 이유는 플랫폼 독립적 인 언어이며 바이트 코드 기능을 통해 보안 및 이식성을 제공한다는 것입니다. 
LINUX, UNIX 및 기타 OS와 같은 다른 플랫폼에서도 작동 할 수 있습니다.
이것은 처음에는 새로운 클래스 나 메소드의 작동 방식을 테스트하기 위해 명령 줄에서 주요 메소드 작성, 재 컴파일 및 프로그램 실행을 더 이상 배우지 않아도되는 초보 학생에게 매우 유용합니다. 교육의 관점에서 볼 때,이 소프트웨어는 "public static void main"과 같은 구문의 의미를 설명하지 않고도 
Java를 작성하는 법을 배우는 매우 쉬운 방법이며 수업에서 데모를 수행하는 데 이상적인 방법을 제공합니다. 
따라서 Java 프로그래밍 언어를 배우고 이해하는 시간이 줄어 듭니다.
주로 사용되는 JAVA 프로그래밍 언어의 패키지
java.awt , java.lang , java.io , java.util , java.net 등이 있다.
